# Executioner Floor 1 Canon

## True Floor 1

The true Spire/Tower Floor 1 is The Executioner.

It is intentionally lethal.

It should not be rewritten as a level-2 tutorial floor.

## The Executioner

Large fiend/demon, chaotic evil.

**Core stats:**

- Armor Class 17
- Hit Points 185
- Speed 30 ft., fly 60 ft.
- Challenge Rating 10
- Proficiency Bonus +4

**Important features:**

- Death Sentence Protocol
- Executioner's Presence
- Magic Resistance
- Legendary Resistance
- Judgment Blade
- Condemning Grasp
- Execution Protocol
- Chains of Judgment
- Legendary Actions
- Lair Actions
- Regional Effects

## Threat Identity

The Executioner punishes:

- isolation
- poor grapple management
- lack of magical weapons
- lack of freedom/mobility tools
- weak healing strategy
- bad positioning
- lack of lightning protection
- rushing the Tower unprepared

## Terminal Intelligence

Player-facing Floor 1 terminal information should include:

- Danger Level: Extreme
- Lives Lost Gathering Intelligence: 47
- Success Rate: 23%
- Threat Designation: The Executioner
- 12-foot humanoid entity
- dark armor-like carapace
- glowing red sensory organs
- oversized blade
- flight capable
- 120-foot circular arena
- lightning strikes
- shifting terrain
- spectral manifestations
- chains
- grapples
- execution countdown
- healing suppression
- death-save danger
- magic resistance
- ACKNOWLEDGE RISK / RETURN prompt

## Design Intention

The terminal should make players seriously consider leaving.

The intended early-campaign response is:

- go to portal worlds
- gain levels
- earn magic items
- earn followers
- improve extraction
- gain tactical intelligence
- return later

## Running Low-Level Attempts

If a low-level party attempts the Executioner anyway:

- warn them clearly
- require explicit risk acknowledgment
- run the real threat
- do not secretly replace it with a tutorial boss
- allow only established retreat/extraction options
- let deaths matter
- make public consequences matter
- let miraculous victory be legendary if it somehow happens
